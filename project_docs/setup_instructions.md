## Prerequisites

- Python 3.8+
- `pip` installed

## Installation

1. Clone the repository:

```bash
git clone <your-repo-url>
cd <your-repo-folder>
```

Create a virtual environment (recommended):

```bash
python3 -m venv venv
source venv/bin/activate
```

Install all requirements:

```bash
pip install -r requirements.txt
```

### RUN poject

```bash
python app.py
```
